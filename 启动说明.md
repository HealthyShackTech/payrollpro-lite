# 项目启动说明

## 方式一：使用启动脚本（推荐）

运行以下命令：

```bash
./START_PROJECT.sh
```

或者：

```bash
bash START_PROJECT.sh
```

## 方式二：手动启动（分两个终端）

### 终端 1 - 启动后端服务器

```bash
cd server
npm start
```

或者：

```bash
cd server
node server.js
```

后端服务器将在 `http://localhost:5001` 启动

### 终端 2 - 启动前端应用

```bash
cd client
npm start
```

前端应用将在 `http://localhost:3000` 启动

## 方式三：使用 npm scripts（如果配置了）

在项目根目录创建 `package.json` 并添加 scripts（可选）

## 访问应用

1. **注册新用户**：访问 `http://localhost:3000/register`
2. **登录**：访问 `http://localhost:3000/login`
3. **使用系统**：登录后访问所有功能

## 检查服务状态

### 检查后端是否运行

```bash
curl http://localhost:5001/health
```

应该返回：`{"ok":true}`

### 检查端口占用

```bash
# 检查后端端口
lsof -i :5001

# 检查前端端口
lsof -i :3000
```

## 停止服务

按 `Ctrl+C` 停止当前终端中的服务

或者使用进程 ID：

```bash
# 查找进程
ps aux | grep "node server.js"
ps aux | grep "react-scripts"

# 停止进程
kill <进程ID>
```

## 常见问题

### 问题：端口已被占用

**解决方案**：
1. 停止占用端口的进程
2. 或者修改 `.env` 文件中的端口配置

### 问题：MongoDB 连接失败

**解决方案**：
1. 检查 MongoDB 是否运行
2. 检查 `server/.env` 中的 `MONGO_URI` 配置

### 问题：依赖未安装

**解决方案**：
```bash
cd server && npm install
cd ../client && npm install
```
